DESCRIPTION
-----------
    DECOY Drone Delivery is a 2D visual simulator displaying multi-threaded
    drones interacting in an obstacle based environment. The implementation
    supports user defined drones. Drones interact using an obstacle detection
    avoidance algorithm.
    
    


INSTRUCTIONS
-----------
    1) Compile by using the make command, 'make'

    2) Run the program using the following command ‘./Decoy’

    3) Enter the number of drones you would like the simulator
       to activate
